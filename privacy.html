Privacy Policy

Effective Date: 2026-05-23

This privacy policy explains how Text Preview Copy Extension handles user data. This extension provides a local text preview and copy feature on web pages based on the user's action.

1. Information We Collect

This extension does not collect, store, sell, or transmit any personal information.

The extension may temporarily access visible text on the current web page only to provide its core feature: showing a local preview of text under the mouse cursor while the Ctrl key is pressed and copying the selected or detected text to the clipboard when the user performs the copy action.

2. How We Use Information

Any page text accessed by the extension is processed locally in the user's browser. The extension uses page text only for displaying a local preview tooltip and copying user-selected or detected text to the clipboard.

3. Data We Do Not Collect

The extension does not:

  • collect browsing history
  • track user behavior
  • read clipboard contents
  • send page text to any external server
  • share data with third parties
  • use data for advertising or analytics

4. Clipboard Access

The extension uses the clipboardWrite permission only when the user explicitly performs a copy action. The extension does not read clipboard contents.

5. Host Permission

The extension uses host access only to run a content script on web pages where the user wants to inspect text. This access is necessary to detect visible text under the mouse cursor and provide the preview and copy feature.

6. Data Storage

This extension does not store user data locally or remotely.

7. Third-Party Sharing

This extension does not share, sell, or transfer user data to third parties.

8. Security

Because the extension does not transmit user data to external servers, page text and copy actions remain local to the user's browser.

9. Changes to This Policy

This privacy policy may be updated when the extension's functionality or data handling practices change.

10. Contact

If you have any questions about this privacy policy, please contact: 0toz@kakao.com
